基本門檻（6 項）
#條件判定邏輯
1近五年財務等級 B 級以上 >= 3 年，且近兩年都 B 級以上B 級以上 = AA/A1/A2/A/B2A/B1A/B1/B2/BA1/BA2（含 +）
帶 - 的（如 B1-）算 B 級以下；C/D/X 也算不過
2累積營收年增率 >= 0%stocks 表 revenue_cum_yoy 欄位
3最佳等級 AA 級依優先順序取第一個有值的：預估 > 系統 > 沈董 > 綜合 > 加權 > 近四季
只要其中一個達到 AA 就通過
4股價低於總表評價 AA收盤價 <= val_aa + 0.005
5綜合殖利率 >= 6%綜合殖利率 = 綜合股利 / 收盤價 × 100
6股利折現模式現價潛在年報酬 >= 10%EPS：預估EPS > 沈董EPS、PE：預設14、股利：綜合股利（三年同值）
年報酬 = ((賣出價+三年股利) / 現價) ^ (1/3) - 1
成長加分（4 項）
#條件判定邏輯
7涅夫保守成長率 >= 7%min(5年CAGR, 3年CAGR) × 折扣係數
涅夫主力投資區為 7~12%；灰色標記的股票不計入
8涅夫 Neff 比率 >= 0.7(保守成長率 + 殖利率) / PE
>= 1.0 極佳、>= 0.7 合格；灰色標記的股票不計入
9林區 PEG <= 1.0PE / (保守成長率 + 殖利率)
林區修正版，將殖利率納入分母；灰色標記的股票不計入
10林區成長一致性 >= 0.51 - |算術平均 - CAGR| / CAGR
< 0.5 視為景氣循環股，不適用 PEG 估值
頂部顯示「基本 X/6 + 加分 Y/4」：基本 >= 5 綠色、>= 3 黃色、< 3 紅色

用途EPS 優先順序股利 優先順序說明
預估等級1. 使用者手動輸入
2. （無則不顯示）		1. 使用者手動輸入
2. （無則不顯示）		個股頁 Q1~Q4 EPS 加總 / 預估現金股利

評價門檻
(AA/A1/A2/A)		1. 預估 EPS
2. 沈董 EPS		1. 預估股利
2. 沈董股利		min(EPS×PE, 股利/殖利率%, 綜合股利/6%+股利)
總表和個股頁的評價門檻使用相同優先順序
沈董等級沈董 EPS沈董股利沈董EPS = 已公佈季度 EPS 加總 × (4/季數)
沈董股利 = 沈董EPS × 平均配息率
綜合等級沈董×權重 + 加權×權重綜合股利權重預設各 50%，可在總表自訂
加權等級五年加權平均加權股利權重 30/25/20/15/10
逍遙價值投資評價法 — EPS / 股利設定邏輯 ▼
欄位自動帶入優先順序說明
EPS設定 1. 使用者手動儲存值
2. Q1~Q4 預估 EPS 加總
3. 沈董 EPS 		開啟個股頁時自動帶入，優先順序由高到低
手動儲存值 = 之前在「EPS設定」欄位手動改過並儲存的值（存在 localStorage/DB vmEps）
Q1~Q4 加總 = 季度估計表的 Q1~Q4 EPS 欄位有值就加總
沈董 = 已公佈季度 EPS 年化
股利設定 1. 使用者手動儲存值
2. 預估現金股利
3. 沈董股利 		同 EPS 邏輯
預估現金股利 = 季度估計表「預估現金股利」欄位
沈董股利 = 沈董EPS × 平均配息率
長期現金股利水準 1. 使用者手動儲存值
2. 綜合股利 		用於計算長期殖利率價格（5%/6%/7%）
綜合股利 = 沈董股利×權重 + 加權股利×權重
偏高殖利率 / 最高殖利率自動帶入從「參數設定」區的個股自訂值讀取
預設：偏高 5.5% / 最高 6%
長期殖利率設定可輸入預設 6%，全部公司共用
影響長期 5%/6%/7% 價格計算的分母
連動機制：季度估計表的 Q1~Q4 EPS 或預估股利變動時，會自動同步更新 EPS設定 和 股利設定
