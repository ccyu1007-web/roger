聶夫總報酬率法 (John Neff) ▼
欄位說明計算方式 / 來源
5年CAGR 過去5年淨利年複合成長率 計算 (最新年度淨利 / 5年前淨利)^(1/5) - 1
使用淨利而非EPS，避免股本變動造成失真
3年CAGR 過去3年淨利年複合成長率（檢查近期趨勢） 計算 (最新年度淨利 / 3年前淨利)^(1/3) - 1
若3年CAGR明顯低於5年CAGR，代表成長正在減速
保守成長率 聶夫「三重驗證」後的保守估計 計算 min(5年CAGR, 3年CAGR) × 折扣係數
折扣係數規則：
兩者差距 ≤ 3%：× 0.8（成長穩定）
差距 3~5%：× 0.75（略有減速）
差距 > 5%：× 0.7（明顯減速，更保守）
Neff比率 總報酬率與本益比的比值，越高越具投資價值 計算 (保守成長率 + 殖利率) / PE
判斷標準：
≥ 1.0：極佳買點（紅色）
0.7 ~ 1.0：合格買點（橘色）
< 0.7：不具吸引力（灰色）
聶夫的取捨原則
成長率範圍聶夫的態度原因
< 7%太慢，純看殖利率被通膨吃掉，成長無意義
7% ~ 12%甜蜜點（主力投資區）可預測、可持續，聶夫最愛
12% ~ 20%可投資但需高度警惕成長可能不持續，要求更便宜的價格
> 20%完全不可預測，避開無法可靠預測，直接放棄不評估
成長品質判斷
品質特徵
高品質既有業務擴張（同店銷售/產能利用率提升）、利潤率改善、市佔率提升、不依賴財務槓桿、不依賴一次性事件
低品質併購灌水（人造成長）、會計政策變更、資產處分（一次性）、財務槓桿（借錢買成長）、減資造成 EPS 提升（股本變動）
聶夫只投資高品質成長的公司。低品質成長即使數字漂亮也不可信。
殖利率來源：預估股利 / 股價（優先用個股自訂，無則用沈董股利）
PE來源：股價 / 預估EPS（優先用個股自訂，無則用沈董EPS）
CAGR > 20% 或成長一致性 < 0.5 的公司會以灰色標記
林區PEG法 (Peter Lynch) ▼
欄位說明計算方式 / 來源
5年CAGR 過去5年淨利年複合成長率（幾何平均） 計算 同聶夫的5年CAGR
林區稱此為「幾何平均」成長率
算術平均 過去5年EPS逐年成長率的簡單平均 計算 Σ(各年EPS YoY成長率) / 年數
例：5年個別成長率為 10%, 15%, 8%, 12%, 20%，算術平均 = 13%
一致性 成長品質指標：算術平均與CAGR的吻合度 計算 1 - |算術平均 - CAGR| / CAGR
判斷標準：
≥ 0.9：極穩定成長，PEG可信度高（綠色）
0.5 ~ 0.9：穩定，正常使用
< 0.5：景氣循環股，不適用PEG（灰色標記）
林區的洞見：算術平均 >> CAGR 時，代表成長波動大，是景氣循環股而非成長股
PEG 本益比成長比，越低代表成長性相對股價越便宜 計算 PE / (保守成長率 + 殖利率)
判斷標準：
≤ 1.0：低估，值得關注（紅色）
1.0 ~ 1.5：合理偏貴（橘色）
> 1.5：偏貴（灰色）
林區修正版：將殖利率加入分母，對高配息公司更公平（傳統PEG = PE / 成長率）
聶夫 vs 林區 的關係
Neff比率 = (保守成長率 + 殖利率) / PE
PEG = PE / (保守成長率 + 殖利率)
兩者互為倒數：Neff比率 = 1 / PEG

差異在於輔助判斷：
聶夫用「5年CAGR vs 3年CAGR」判斷成長是否減速
林區用「算術平均 vs CAGR」判斷成長是否穩定（排除景氣循環股）
聶夫/林區 灰色標記說明 ▼
什麼是灰色標記？
灰色標記代表「數字算得出來，但方法論本身認為不適用這支股票」。
即使 Neff 比率或 PEG 的數字看起來符合標準，灰色標記提醒你這個結果要打問號。
灰色條件原因來自誰的規則
5年CAGR > 20% 聶夫認為超過 20% 的成長率不可持續、無法可靠預測，直接放棄不評估 聶夫（John Neff）
「成長率超過 20%，無法可靠預測，直接放棄。」
成長一致性 < 0.5 算術平均成長率和 CAGR 差距太大，代表 EPS 波動劇烈，是景氣循環股而非成長股，PEG 估值無意義 林區（Peter Lynch）
「算術平均看起來像高成長，但 CAGR 很低——這是景氣循環股，不是成長股！不能用 PEG 估值。」
範例
股票情境數字結果
A公司 5年CAGR = 8%，一致性 = 0.95 Neff = 1.2，PEG = 0.83 正常顯示，可照數字判斷
B公司 5年CAGR = 27%，一致性 = 0.86 Neff = 0.53，PEG = 1.88 灰色標記（CAGR > 20% 不可信）
C公司 5年CAGR = 15%，一致性 = 0.35 Neff = 0.9，PEG = 1.1 灰色標記（景氣循環股，PEG 無意義）
簡單記法：白色 = 可以照數字判斷，灰色 = 方法論本身說不適用這支股票
